# 우리만의 디데이
- 기존 디데이 어플에서 달력을 추가해 일정을 관리해주는 앱
- 기간 : 2022.02.17 ~ 2022.03.28
- 기능
  - 디데이 계산 기능
  - 기념일 계산 기능 (100일, 1년 단위)
  - 일정 관리 기능
  - 홈 화면 위젯 지원
- 라이브러리
  - Realm
  - FSCalendar
  - CropViewController
  - IQKeyboardManagerSwift
